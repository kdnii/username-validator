# username-validator
---

- Username cannot be more than **12 characters**  
- Username must **not contain spaces**  
- Username must **only contain alphabet letters** (no digits or symbols)  

---
# Example Output

```
Enter a username: meow1$23
ONLY ALPHABET!
```

```
Enter a username: meow meow
Your username cannot contain spaces
```

```
Enter a username: veryloudmeowwwww
Your username cannot be more than 12 characters
```
